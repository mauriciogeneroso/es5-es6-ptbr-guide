## 12 Coleções (Map, Set)

As classes de collections Map e Set foram instroduzidas no Javascript na versão ES6. São estruturas de dados clásicas e que podem ser visualizadas nos arquivos de exemplo.

* O Map é um mapa de chave e valor. Caso seja inserido mais de um valor para a mesma chave, será mantido a chave e o valor será atualizado.
* O Set não permite valores duplicados.
